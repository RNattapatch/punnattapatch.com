คำตอบคือ "ได้" ลูกค้าผมส่วนใหญ่ไม่มีใครเขียน code และนั่นคือเหตุผลที่ผมเลือก stack แบบนี้

## ตัวอย่างจริง

ทีมขาย Forklift 5 คน ไม่มีใครเก่ง tech เคยทำ report ให้หัวหน้าสัปดาห์ละ 2 ชั่วโมง ตอนนี้ใช้ AI workflow ที่ผมวางให้ เหลือ 15 นาทีต่อสัปดาห์

## ทำไมคุณไม่ต้องเก่ง tech

ผมออกแบบระบบให้ทีม non-tech maintain ต่อเองได้ ด้วย 3 เครื่องมือนี้

- **Google Sheets** — ทีมใช้เป็นอยู่แล้ว ไม่ต้องเรียนใหม่
- **n8n** — ลาก-วางเอา workflow มาต่อกัน ไม่ต้องเขียน code
- **Google Apps Script** — ใช้ตอน logic ซับซ้อนเท่านั้น และผมเขียนให้เสร็จเป็น template ที่ทีมก๊อปไปใช้ได้เลย

## สิ่งที่ทีมต้องทำเป็น (แค่นี้พอ)

- เปิด Google Sheets แล้วแก้ข้อมูลได้
- กด approve email ตอน n8n ขอ permission ครั้งแรก
- อ่าน LINE notification ที่ workflow ส่งมา

เท่านี้พอ — ที่เหลือผมวางให้หมด พร้อม screen recording ที่เปิดดูซ้ำได้ทุกเมื่อ

## ข้อแม้อย่างเดียว

ทีมต้องเปิดใจลองของใหม่ — ถ้ามี mindset ว่า "AI จะมาแย่งงาน" หรือ "ทำแบบเดิมดีอยู่แล้ว" ระบบที่ผมวางให้ก็จะเงียบใน 2 อาทิตย์

ผมเจอเคสนี้ไม่บ่อย แต่เคยเจอ — ถ้าเป็นเคสนี้ ผมบอกตรงๆ ว่ายังไม่ควรลงทุน ควรรอให้ทีมเห็นผลจากธุรกิจอื่นก่อน
