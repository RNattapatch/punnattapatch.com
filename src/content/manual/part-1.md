## "AI ถามตอบ" vs "AI ทีมงาน"

คุณอาจเคยลองใช้ ChatGPT หรือ AI ตัวอื่นๆ มาบ้างแล้ว ส่วนใหญ่เป็นการถาม-ตอบเป็นครั้งๆ ไป เหมือนคุยกับคนแปลกหน้าบนถนน คุณถามทาง เขาก็ตอบ แต่พอคุณเดินไปอีก 2 บล็อกแล้วถามคำถามใหม่ เขาจำไม่ได้แล้วว่าเคยคุยอะไรกับคุณ

นั่นคือ "AI ถามตอบ"

มันมีประโยชน์สำหรับงานเล็กๆ เฉพาะหน้า แต่สำหรับธุรกิจ B2B ที่มีความซับซ้อน เราต้องการมากกว่านั้น เราต้องการ "AI ทีมงาน"

AI ทีมงาน หรือที่ผมเรียกว่า AI Agent คือระบบ AI ที่ถูกสร้างขึ้นมาโดยมี "ความจำ" และ "บริบท" เกี่ยวกับธุรกิจของคุณโดยเฉพาะ มันไม่ใช่แค่การ prompt ให้เก่งขึ้น แต่คือการสร้างพนักงานดิจิทัลที่รู้จักสินค้า บริการ ลูกค้า และเป้าหมายของบริษัทคุณ

ลองดูตารางเปรียบเทียบนี้ครับ

| คุณสมบัติ            | AI ถามตอบ (ทั่วไป)                         | AI ทีมงาน (Agent)                                                                                                    |
| :------------------- | :----------------------------------------- | :------------------------------------------------------------------------------------------------------------------- |
| **ความจำ (Memory)**  | จำได้แค่ใน session ที่คุย ล้างใหม่ทุกครั้ง | จำข้อมูลธุรกิจ, template, และ interaction ที่ผ่านมาได้                                                               |
| **บริบท (Context)**  | ไม่มีบริบทเกี่ยวกับบริษัทของคุณ            | ถูกสร้างมาพร้อมบริบท: สินค้าคืออะไร, ลูกค้าคือใคร, แบรนด์ของเราเป็นแบบไหน                                            |
| **เป้าหมาย (Goal)**  | ให้คำตอบตามสิ่งที่คุณถาม                   | ทำงานตามเป้าหมายที่ตั้งไว้ให้สำเร็จ เช่น "ร่างอีเมล follow-up" "วิเคราะห์ข้อมูลลูกค้าที่ทัก Chat มา ทุกวันตอน 16.00" |
| **ผลลัพธ์ (Output)** | ต้อง copy ข้อความไป paste ในไฟล์เอง        | ผลลัพธ์ที่เกือบพร้อมใช้งานได้ทันทีตาม format ที่กำหนด                                                                |

AI Agent ไม่ได้รอให้คุณป้อนคำสั่งทุกครั้ง แต่มันถูกออกแบบมาให้มี "บทบาทหน้าที่" (Role) ชัดเจน เช่น Agent A ทำหน้าที่รีเสิร์ชข้อมูลบริษัทเป้าหมาย, Agent B ทำหน้าที่ร่างอีเมลเปิดการขาย, Agent C ทำหน้าที่สรุปประชุมจากไฟล์เสียง

เมื่อคุณมีทีม AI Agent แต่ละตัวจะทำงานประสานกันเหมือนทีมงานคนหนึ่ง มันลดเวลาที่คุณและทีมต้องใช้ไปกับงาน routine ที่ไม่สร้างรายได้ ทำให้ทีมขายของคุณมีเวลาไปโฟกัสกับสิ่งที่สำคัญที่สุด—การสร้างความสัมพันธ์และปิดการขาย

> โลกทุกวันนี้มันหมุนเร็วขึ้นมากๆ AI อัพเดตทุกวัน ถ้าคุณยังใช้เครื่องมือ AI แค่ตัวที่คุณ "ชินมือ" ข่าวร้ายคือคุณอาจจะตกขบวนรถไฟแห่ง Agentic AI Transformation แล้ว
>
> สิ่งที่ผมเรียบเรียงและเขียนออกมาเป็นคู่มือนี้ มาจาก Pain Point ที่ผมได้เข้าไปช่วย Consult ธุรกิจของ SME ไทยหลายๆ ราย ทั้งปัญหาคลาสสิคที่ต้องเจอ
> - ทีมไม่กล้าใช้ AI กลัว Prompt ผิด
> - ใช้ AI เป็นแค่เขียนเมลล์ แปลภาษา เขียนประกาศรับสมัครงาน หรือใช้ทำรูปหมาแมวตลกๆ น่ารักๆ
> - เจ้าของอยากพัฒนาเรื่อง AI แต่ไม่มีเวลาสอนพนักงาน
> - พนักงานขายที่ต้องเสียเวลานั่งพิมพ์ Report แทนที่จะเอาเวลาไปดูแลลูกค้าหรือหายอดขายเพิ่ม
>
> คู่มือเล่มนี้จะสอนคุณสร้าง "AI ทีมงาน" ไม่ใช่แค่สอนวิธีใช้ "AI ถามตอบ" ครับ

## ROI case — เคสลูกค้าที่ผมดูแลในไตรมาสที่ผ่านมา

ทฤษฎีฟังดูดี แต่ทำได้จริงไหม?

ผมขอยกเคสลูกค้าที่ผมดูแลในไตรมาสที่ผ่านมา เป็นธุรกิจ B2B ขนาดกลางในกลุ่ม SME ที่ผมดูแล ปัญหาก่อนหน้านี้คือทีมขายใช้เวลาเยอะมากไปกับงานที่ไม่ใช่การขายโดยตรง เช่น การหาข้อมูลบริษัทลูกค้าเป้าหมาย การเตรียมข้อมูลก่อนเข้าประชุม และการร่างอีเมลติดตามผลหลังประชุม

งานเหล่านี้กินเวลาเฉลี่ย 1-2 ชั่วโมงต่อเซลส์ 1 คนในทุกๆ วัน

สิ่งที่ผมเข้าไปทำคือการสร้าง AI Agent ขึ้นมาเพียง 1 ตัว เพื่อรับผิดชอบงาน 3 อย่างนี้โดยเฉพาะ:

1. **Research Agent:** รับแค่ชื่อบริษัท แล้วไปค้นหาข้อมูลสำคัญ เช่น ขนาดบริษัท, ผู้บริหาร, ข่าวล่าสุด, งบการเงินเบื้องต้น แล้วสรุปมาเป็น bullet point 5 ข้อ
2. **Pre-Meeting Brief Agent:** รับข้อมูลสรุปจาก Research Agent แล้วร่างเอกสารสรุป 1 หน้าสำหรับให้เซลส์อ่านก่อนเข้าประชุม
3. **Follow-up Draft Agent:** รับโน้ตสั้นๆ จากเซลส์หลังประชุม แล้วร่างอีเมลติดตามผลลูกค้าตาม template ของบริษัท

เราใช้เวลาเซ็ตอัพ Agent ตัวนี้ประมาณครึ่งวัน และให้ทีมขายเริ่มใช้งานจริงในสัปดาห์ถัดมา

ผลลัพธ์ที่เกิดขึ้นชัดเจนมากครับ

| | ก่อนใช้ AI Agent | หลังใช้ AI Agent (1 ตัว) |
| :--- | :--- | :--- |
| **เวลาของทีมขาย** | ใช้เวลา 20-30% ไปกับงาน admin และ research | เวลา admin ลดลงเหลือไม่ถึง 5% · มีเวลาโทรหาลูกค้าและเข้าประชุมเพิ่มขึ้น |
| **คุณภาพงาน** | ข้อมูล research ไม่สม่ำเสมอ · อีเมลติดตามผลช้า | ข้อมูลมีมาตรฐานเดียวกัน · ส่งอีเมล follow-up ได้ภายใน 1 ชั่วโมงหลังประชุม |
| **ผลลัพธ์ทางธุรกิจ** | ปิดดีลได้ตามเป้า แต่ทีมเริ่มหมดไฟ | ทีมขายมี morale ดีขึ้น · เอาเวลาไปสร้าง pipeline ใหม่ๆ จนสามารถปิดดีล workshop และ consulting เพิ่มเติมได้อย่างต่อเนื่อง |

เคสนี้คือผลลัพธ์จากการสร้าง Agent แค่ 1 ตัว ที่ทำงานเฉพาะทางมากๆ แต่ช่วยปลดล็อกเวลาของทีมงานทั้งทีมได้มหาศาล

> อีกเคสนึงของลูกค้าผม เจ้าของพยายามเอา KPI ที่ ChatGPT คิดให้มาใช้งานจริง แต่พนักงานไม่เอาด้วย เพราะรู้สึกว่า 'ทำจริงไม่ได้ ใช้จริงไม่ได้' ตั้งเป้าวัดผลแบบเป็นไปไม่ได้ แถมยังต้องมานั่งกรอกข้อมูลทุกวันตอน 17.00 ตามที่ ChatGPT บอก?
>
> เจ้าของเขาเลยมาปรึกษาผมว่าทำอย่างไรดี ผมเลยแนะนำว่าลองสร้าง KPI โดยให้น้องๆ ในทีมมีส่วนร่วมในการออกแบบด้วย และเอาเป้าที่เจ้าของอยากได้มา Fusion เข้าด้วยกัน จากนั้นทำระบบ Automation ที่จะวิ่งไปเช็คการทำงานของน้องๆ ทุกวันเอง ไม่ต้องให้น้องมากรอก โดย Automation เจ้าของเขาก็ไม่ได้เขียน code เองแม้แต่บรรทัดเดียว แต่ให้ AI Agent ที่ผมแนะนำ สร้างระบบขึ้นมาให้
>
> ผลลัพธ์คือได้ระบบ KPI ฝ่ายขายที่ใช้ได้จริง วัดผลได้ มี Data ไปพัฒนายอดขาย โดยที่ไม่ต้องให้น้องๆ มาเสียเวลานั่งกรอกข้อมูล

## ทำไม SME ไทยต้องเริ่มวันนี้

คำถามที่ผมเจอบ่อยคือ "ทำไมต้องรีบ?" "รอให้เทคโนโลยีมันนิ่งกว่านี้ก่อนดีไหม?"

คำตอบของผมคือ ต้นทุนของการ "ไม่ทำอะไรเลย" สูงกว่าที่คุณคิดมากครับ

ลองคิดถึงงาน admin หรืองาน routine ที่เกิดขึ้นในบริษัทของคุณทุกวัน งานที่กินเวลาแต่ไม่สร้างรายได้โดยตรง เช่น การคีย์ข้อมูล, การทำรีพอร์ต, การร่างเอกสาร, การตอบคำถามซ้ำๆ งานเหล่านี้คือ "ต้นทุนค่าเสียโอกาส" ที่คุณจ่ายอยู่ทุกวันโดยไม่รู้ตัว

**Tier 1: เจ้าของธุรกิจทำเอง หรือมีทีม 1-2 คน**

สมมติว่าคุณทำงาน routine พวกนี้เอง สัปดาห์ละ 5 ชั่วโมง ถ้าคิดค่าแรงของคุณแบบ conservative ที่ชั่วโมงละ 500 บาท

5 ชั่วโมง/สัปดาห์ × 50 สัปดาห์/ปี × 500 บาท/ชั่วโมง = **125,000 บาท/ปี**

**Tier 2: ทีมเล็ก 5-10 คน**

สมมติว่าแต่ละคนในทีมใช้เวลากับงาน admin คนละ 2 ชั่วโมง/วัน หรือ 10 ชั่วโมง/สัปดาห์ สำหรับทีม 8 คน นั่นคือ 80 ชั่วโมง/สัปดาห์

80 ชั่วโมง/สัปดาห์ × 50 สัปดาห์/ปี × 500 บาท/ชั่วโมง = **2,000,000 บาท/ปี**

**Tier 3: ทีมขนาด 10-20 คน**

ถ้าทีมคุณใหญ่ขึ้น ปริมาณงาน admin ก็จะยิ่งสูงตามไปด้วย สมมติว่ามีงาน routine ที่ต้องทำรวมกันทั้งบริษัท 240 ชั่วโมงต่อสัปดาห์

240 ชั่วโมง/สัปดาห์ × 50 สัปดาห์/ปี × 500 บาท/ชั่วโมง = **6,000,000 บาท/ปี**

ตัวเลขพวกนี้ไม่ใช่เงินที่คุณจะ "ประหยัดได้" ในอนาคต—แต่มันคือต้นทุนที่คุณจ่ายอยู่แล้วทุกวันนี้

> **AI มาได้ไม่ใช่มาแทนที่คน แต่ AI ทำให้ช่องว่างของคนที่ 'ลงมือทำ' กับคนที่รอให้พร้อมแล้วค่อยทำ ห่างกันมากขึ้นจนแทบตามไม่ทัน ในขณะที่คุณรอให้พร้อม คนที่เขาลงมือทำจริงเค้าใช้ AI ลองผิดลองถูกจนได้ผลลัพธ์ที่ Success ไปแล้ว 'อย่ารอให้พร้อม ต้องเริ่มวันนี้ทันที'**

## Hybrid human-AI org chart

AI ไม่ได้มาแทนที่คนเก่งของคุณ แต่มันมาแทนที่ "งานน่าเบื่อ" ที่คนเก่งของคุณไม่อยากทำ

หลักการแบ่งงานง่ายมากครับ:
- **AI Agent ทำงาน 80%** — งานที่มีขั้นตอนซ้ำๆ, งานที่เกี่ยวกับข้อมูล, งานที่ต้องใช้ความเร็วและความแม่นยำสูง
- **มนุษย์ (ทีมของคุณ) ทำงาน 20%** — งานที่ต้องใช้ความคิดสร้างสรรค์, การสร้างความสัมพันธ์, การเจรจาต่อรอง, การตัดสินใจเชิงกลยุทธ์

**งานที่ AI Agent ควรทำ:**
- รีเสิร์ชข้อมูลลูกค้าจากเว็บไซต์และข่าว
- สรุปการประชุมจากไฟล์เสียงหรือ transcript
- ร่างอีเมลฉบับแรก (cold email, follow-up) ตาม template
- ดึงข้อมูลจากระบบ CRM มาทำรีพอร์ตเบื้องต้น
- คีย์ข้อมูลจาก Chat Line ไปใส่ใน Excel อัตโนมัติ
- สรุปยอดขายรายวันส่งเจ้านาย

**งานที่คน (Human) ควรทำ:**
- โทรศัพท์หาลูกค้าเพื่อสร้างความสัมพันธ์
- เข้าประชุมเพื่อทำความเข้าใจปัญหาของลูกค้าจริงๆ
- นำเสนอ solution และเจรจาต่อรอง
- ตัดสินใจว่าจะให้ส่วนลดหรือไม่
- วางกลยุทธ์การขายสำหรับไตรมาสถัดไป

ทีมขาย 5 คนของคุณ จะทำงานได้เหมือนทีม 15 คน ไม่ใช่เพราะพวกเขาทำงานหนักขึ้น แต่เพราะพวกเขาได้ "เลิกทำ" งานที่ไม่ใช่การขายไปแล้ว 80%

## The "winning zone" — ทำไม Kit นี้ต่างจาก free AI course อื่น

ความรู้ในคู่มือนี้มาจาก "winning zone" ของผม ซึ่งคือจุดตัดของประสบการณ์ 4 ด้าน:

1. **5 ปีในสาย B2B Sales:** ผมเริ่มต้นจากการเป็นเซลส์ B2B ลงมือทำธุรกิจเอง ผมเข้าใจดีว่าวันๆ หนึ่งของทีมขายและเจ้าของธุรกิจต้องเจอกับอะไรบ้าง

2. **ประสบการณ์ปั้นทีมขาย 100 ล้าน:** ผมเคยสร้างและบริหารทีมขายที่ทำยอดขายระดับ 9 หลักมาแล้ว รู้ว่าต้องปั้นทีมขายยังไง พัฒนาฝ่ายขายอย่างไร

3. **ใช้ AI ทุกวันด้วยตัวเอง:** ผมไม่ได้แค่สอนทฤษฎี ผมใช้เครื่องมือเหล่านี้ทุกวันในการทำงานของผม ตั้งแต่การหาลูกค้า, การตลาด, ไปจนถึงการจัดการเอกสารหลังบ้าน

4. **บริหารธุรกิจตัวเองโดยไม่มีพนักงาน:** ธุรกิจที่ปรึกษาของผมเองรันโดยไม่มีพนักงานประจำแม้แต่คนเดียว ผมใช้ AI Agent กว่า 20 ตัวทำงานแทนในส่วนต่างๆ ผมพิสูจน์แล้วว่าโมเดลนี้มันเวิร์ค
