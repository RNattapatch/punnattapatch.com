## ทำไม 1 วัน ไม่ใช่ 3 เดือน

คอร์ส AI ทั่วไปให้เวลา 2–3 เดือน แล้วหวังให้คุณลงมือเอง P1 ไม่ทำแบบนั้น เพราะรู้ว่ามันไม่ work

P1 ใช้ 3 กติกาที่ทำให้ 1 วันพอ:

**1. โฟกัส 1 use case เท่านั้น** — คุณเลือก 1 งานที่กินเวลาทีมมากที่สุด แล้วสร้าง Agent สำหรับงานนั้นก่อนกลับบ้าน ไม่ใช่แผนงานบนกระดาษ

**2. ไม่เขียน code สักบรรทัด** — เราใช้ n8n + ChatGPT ซึ่งเจ้าของธุรกิจดูแลและแก้เองได้โดยไม่ต้องรอ dev

**3. Peer-accountability บังคับ** — ทุกคนต้องนำเสนอ Agent ที่สร้างใน Module 4 ไม่มีใครกลับบ้านมือเปล่า
