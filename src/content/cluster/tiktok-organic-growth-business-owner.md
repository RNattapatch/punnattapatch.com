ผมเปิดช่อง TikTok ใหม่ ใช้เวลา 24 วัน ขึ้นครบ 10,000 followers ไม่ยิง ads แม้แต่บาทเดียว ปิด deal consulting ได้จริง มี sponsor inbound เข้ามา

นี่คือสิ่งที่เจ้าของ SME กับทีมขายเอาไปทำตามได้ทันที 6 ขั้นตอนที่ผมใช้จริง ไม่ใช่ theory

## Part 1: เลือกกลุ่มคนดูก่อนลงมือถ่ายคลิปแรก

ก่อนถ่ายคลิปต้องตอบคำถามนี้ให้ได้ "ใครคือคนที่อาจจะใช้สินค้าหรือบริการของเรา"

ห้ามตอบกว้างเช่น "เจ้าของธุรกิจทั่วไป" ต้องเจาะจงพอที่เราจะนึกออกว่าเขาดูคลิปอะไรอยู่

วิธีคิด 3 ชั้น
- Layer 1 คือใครที่ซื้อจริง (พนักงานที่ตัดสินใจ หรือเจ้าของ)
- Layer 2 คือ pain อะไรที่ทำให้เขา search คลิปแบบของเรา
- Layer 3 คือ value ที่เขาได้จากการดูคลิปล้วนๆ (ยังไม่เกี่ยวกับการซื้อของ)

ตัวอย่างช่องผม
- Layer 1: เจ้าของธุรกิจ SME ฿30-200M ที่อยากได้ inbound lead
- Layer 2: pain คือยอดคงที่ ทีมขายไม่มีระบบ commission ไม่ fair
- Layer 3: value คือ framework ที่เอาไป test ในทีมได้เลย ไม่ต้องจ้าง consultant

ถ้าตอบ 3 layer นี้ไม่ได้ หยุดอย่าถ่ายคลิป กลับไปคิดก่อน

## Part 2: สร้างตัวตนของช่องให้ชัดก่อนขายของตรงๆ

TikTok algorithm เกลียดคอนเทนต์ขายของตรงๆ และลูกค้าไม่ซื้อของจากคนแปลกหน้า เขาจะซื้อจากคนที่รู้สึกว่ารู้จักและไว้ใจ

Identity framework 3 layer ที่ทำให้คนจำได้
1. เอาสิ่งที่ลูกค้าอยากรู้เป็นตัวตั้ง แทนที่จะพูดแต่เรื่องที่เราอยากพูด
2. หาจุดเด่นที่คนอื่นไม่มี ผสมกับความชอบของผู้ชม ไม่ต้องรวย หล่อ สวย เช่น ชอบแมว ไม่ชอบกินข้าวแกง พูดไทยปนอังกฤษ
3. ใส่ตัวตนเข้าไปในคอนเทนต์ ทำซ้ำเรื่อยๆ ให้คนจำตัวเราได้

ของผมคือ "ปัน นักปรึกษาการขายที่ชอบส้ม" — expertise + quirk ใช้ด้วยกันทุกคลิป จน audience จำได้

## Part 3: แต่งตัวก่อนลงสนาม

ผู้ใช้ TikTok ใช้เวลาแค่ 0.8-1.5 วินาทีตัดสินใจว่าจะดูคลิปต่อหรือปัดทิ้ง ทุกอย่างในโปรไฟล์ของคุณเลยต้องช่วยกันดึงคนไว้

รูปโปรไฟล์ ต้องชัดเจน เห็นหน้าแล้วรู้เลยว่าคุณเป็นใคร ทำอะไร หลีกเลี่ยงรูปส่วนตัวที่ไม่เกี่ยวข้องกับงาน เช่น รูปเที่ยวทะเลหรือรูปสัตว์เลี้ยง

Bio ต้องอ่านแล้วรู้ทันทีว่าคุณช่วยอะไรเขาได้ ตัวอย่าง
- ✅ "เซลล์ขายรถที่ดูแลลูกค้าจบสุด ส่งมอบแล้วยังติดต่อได้ 24 ชม."
- ✅ "ผมช่วยคนแก้ปัญหานอนไม่หลับ โดยไม่ใช้ยา"
- ❌ "เลี้ยงหมา รักการเที่ยว ชอบฟังเพลง" — ไม่มี value proposition

คลิปปักหมุด 3 คลิปแรก ให้เลือกคลิปที่
- เล่าเรื่องตัวคุณ (who you are)
- สื่อตัวตนที่ unique (what makes you different)
- แสดงให้เห็นว่าคุณแก้ปัญหาอะไรให้ใคร (what you do)

## Part 4: เขียนบทคอนเทนต์ให้ตรงกับ keyword ของช่อง

ก่อนหยิบมือถือขึ้นมาถ่าย ต้องออกแบบคอนเทนต์ก่อนทุกครั้ง

5 คำถามที่ต้องตอบก่อนกด Record
1. Content นี้เป้าหมายคืออะไร (growth / nurture / convert)
2. Keyword หลักของคลิปคือคำไหน ที่ audience พิมพ์ search จริง
3. เปิดหัวยังไงให้หยุดคนดูใน 3 วิแรก (specific claim / counter-intuitive / direct question)
4. ทำยังไงให้คนดูมีเป้าหมายในการดูจนจบ (curiosity loop / promise / cliffhanger)
5. จบคลิปแล้ว audience จะรู้สึก/ทำอะไรต่อ (comment / save / follow / DM)

ตอบ 5 ข้อนี้ให้ครบก่อน แล้วค่อยเริ่มถ่าย

เทคนิคเปิด 3 วิที่ใช้บ่อยในช่องผม
- Specific claim: "ยอดขายคงที่ทุกปีนี่สัญญาณอันตราย"
- Counter-intuitive: "จ้างเซลล์ก่อนมีระบบ = burn out ใน 3 เดือน"
- Direct question: "ร้านคุณตอบลูกค้าภายในกี่นาที"

## Part 5: วัดผลแบบที่เอาไปปรับปรุงต่อได้

Followers เป็น vanity metric. Metric ที่ควรวัดทุก 7 วัน

| Metric | วัดอะไร | Target ช่วงแรก |
|---|---|---|
| Watch time เฉลี่ย | คน retain อยู่นานไหม | ≥ 50% ของความยาวคลิป |
| Completion rate | ดูจบไหม | ≥ 30% |
| Share rate | คลิปน่า share หรือไม่ | ≥ 1% ของ views |
| Profile visit / view | คลิปดึงคนมาที่ profile ไหม | ≥ 2% |
| Follow / profile visit | Profile convince คนให้ follow ไหม | ≥ 10% |
| DM / follower | Follower กลายเป็น lead ไหม | ≥ 0.3% |

ถ้าตัวเลขต่ำกว่า target ต้องรู้ว่าต้องปรับจุดไหน
- Watch time ต่ำ = hook หรือ middle section พัง
- Profile visit ต่ำ =CTA ท้ายคลิปไม่กระตุ้น
- Follow rate ต่ำ = bio หรือ pinned clips ไม่ convince
- DM rate ต่ำ = ไม่มี funnel ที่ชวนคนคุยต่อ

## Part 6: เปลี่ยนคนดูเป็นลูกค้า

Followers เป็นสินทรัพย์ แต่ยังไม่ใช่รายรับ ต้องมีระบบเปลี่ยนให้เป็นดีล

3 หลักที่ผมใช้ทุกคลิป
- สร้างคอนเทนต์ที่ให้ประโยชน์กับคนดูจริงๆ ไม่ว่าจะเป็นข้อมูลที่จับต้องได้ (framework, checklist, ตัวเลข) หรือความรู้สึกร่วม (เรื่องเล่าที่โดนใจ, ความเข้าอกเข้าใจ)
- ทำให้คนดูจำได้ว่าเราช่วยแก้ปัญหาอะไรให้เขา หรือถ้าติดตามช่องแล้วจะได้อะไร
- ทำซ้ำสม่ำเสมอ เน้นให้ประโยชน์มากกว่าพยายามขายของตรงๆ

Conversion funnel ที่ผมใช้กับช่องตัวเอง
1. Comment ที่เกี่ยวกับปัญหา → reply เชิญมา DM
2. DM → template 3 คำถาม qualify ใน 15 นาที (budget / timeline / decision process)
3. Qualified → นัด discovery call 30 นาที ฟรี
4. Call → ถ้า fit ส่ง proposal ฿30,000-฿65,000 fixed-price ใน 24 ชม.
5. Close หรือ disqualify ภายใน 7 วัน ไม่ไล่ยืด

## 5 ข้อผิดพลาดที่เจ้าของ SME ทำตอนเปิดช่องใหม่

1. เปิดช่องแล้ว copy คนดังโดยตรง ไม่มี identity ของตัวเอง
2. โพสต์ขายของตั้งแต่คลิปแรก algorithm ไม่ push คนไม่ follow
3. ไม่มี content calendar ไม่มี pillar ชัด โพสต์ตามอารมณ์ล้วนๆ
4. ด่วนตัดสินว่าช่องไม่เวิร์คใน 2 อาทิตย์แรก (เร็วเกินไป เพราะ algorithm ใช้เวลาทดสอบ 3-6 สัปดาห์)
5. ไม่ set funnel convert followers → lead → customer จบที่ vanity metric
