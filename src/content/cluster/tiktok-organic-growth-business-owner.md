ผมเปิดช่อง TikTok ใหม่วันที่ 22 ธันวาคม 2568 ใช้เวลา 24 วันได้ครบ 10,000 followers ไม่ยิง ads แม้แต่บาทเดียว ปิดดีล consulting ได้จริง มี sponsor `inbound (ลูกค้าที่ทักมาเอง)` เข้ามา

บทความนี้ไม่ใช่เรื่อง "ทำยังไงให้มียอดฟอล" แต่เป็นเรื่อง funnel analytics ที่ทำให้ช่อง TikTok ปิดดีล B2B ได้จริง + วิธีใช้ AI เสริม production โดยไม่ทำลาย trust

## ตัวเลขจริงจากช่อง @pun_nattapatch (ข้อมูล 2568-12-22 ถึง 2569-04-19)

| Metric | ค่าจริง | หมายเหตุ |
|---|---|---|
| Total followers | 18,812 | เริ่ม 0 เมื่อ 4 เดือนก่อน |
| Total video views | 2.29M | จากคลิปทั้งหมด 15 คลิป |
| Avg views/clip | ~132,000 | มี viral 2 คลิป (500k+) |
| Profile visit rate | 0.99% | ของ total views (key funnel metric) |
| Share rate | 1.55% | ของ total views |
| Engagement rate | 7.9% | (like + comment + share) / views |
| Audience | TH 87%, Female 63% | target เจ้าของ SME ไทย |
| Ads spend | ฿0 | organic ทั้งหมด |

ตัวเลขเหล่านี้ไม่ใช่ `vanity metric (ตัวเลขสวยๆ ที่ไม่สร้างรายได้)` แต่เป็น leading indicator ของ funnel ที่ปิดดีลได้

## Funnel Model: จาก View → ดีล B2B

| Layer | ตัวอย่างช่องผม | Conversion rate | ต้อง optimize อะไร |
|---|---|---|---|
| 1. Video view | 2,287,644 views | — | Hook 3 วิแรก + keyword ตรงกลุ่ม |
| 2. Profile visit | 22,658 visits | 0.99% ของ views | CTA ท้ายคลิป + hook ชื่อ/bio ที่ดึงคนเข้า profile |
| 3. Follow | 18,812 followers | 83% ของ profile visits | Bio + pinned clips ที่ convince |
| 4. DM inbound | รายใหม่ทุกวัน | — | Content ที่ชวนคุยต่อ (comment → DM) |
| 5. Discovery call | qualified leads จาก DM | — | Template 3 คำถาม qualify ใน 15 นาที |
| 6. Close ดีล | consulting + sponsor | — | Proposal fit + F/U ใน 7 วัน max |

Layer 1-3 วัดได้จาก TikTok Studio. Layer 4-6 ต้อง track ใน CRM ของตัวเอง (ผมใช้ Google Sheets + Apps Script)

ถ้า layer ไหน conversion ต่ำกว่า target ต้องแก้ที่ layer นั้นตรงจุด ไม่ใช่โพสต์เพิ่ม

## 6 ขั้นตอนที่ผมใช้จริง

### Part 1: เลือกกลุ่มคนดูก่อนลงมือถ่ายคลิปแรก

ก่อนถ่ายคลิปต้องตอบคำถามนี้ให้ได้ "ใครคือคนที่อาจจะใช้สินค้าหรือบริการของเรา"

ห้ามตอบกว้างเช่น "เจ้าของธุรกิจทั่วไป" ต้องเจาะจงพอที่เราจะนึกออกว่าเขาดูคลิปอะไรอยู่
ชอบคลิปแบบไหน
ยิ่งเราบอกว่าได้ว่าเขาจะเป็นคนแบบไหน
อายุ เท่าไหร่ เพศอะไร มีความกลัว หรือเป้าหมายเรื่องอะไร
ยิ่งเพิ่มโอกาสการทำ content ให้ตรงโจทย์ของกลุ่มได้มากขึ้นครับ

วิธีคิด 3 ชั้น
- Layer 1 คือใครที่ซื้อจริง (พนักงานที่ตัดสินใจ หรือเจ้าของ)
- Layer 2 คือ pain อะไรที่ทำให้เขา search คลิปแบบของเรา
- Layer 3 คือ value ที่เขาได้จากการดูคลิปล้วนๆ (ยังไม่เกี่ยวกับการซื้อของ)

ตัวอย่างช่องผม
- Layer 1: เจ้าของธุรกิจ SME ฿30-200M ที่อยากได้ `inbound lead (ลูกค้าที่สนใจที่ทักเข้ามาเอง)`
- Layer 2: pain คือยอดคงที่ ทีมขายไม่มีระบบ `commission (ค่าคอมมิชชั่น)` ไม่แฟร์
- Layer 3: value คือ framework ที่เอาไป test ในทีมได้เลย ไม่ต้องจ้าง `consultant (ที่ปรึกษา)`

ถ้าตอบ 3 layer นี้ไม่ได้ หยุดอย่าถ่ายคลิป กลับไปคิดก่อน

### Part 2: สร้าง identity ที่ลูกค้าจำได้ก่อนขายของ

TikTok `algorithm (ระบบคัดเลือกคลิป)` เกลียดคอนเทนต์ขายของตรงๆ และลูกค้า B2B ไม่ซื้อของจากคนแปลกหน้า เขาจะซื้อจากคนที่รู้สึกว่ารู้จักและไว้ใจ

Identity framework 3 layer ที่ทำให้คนจำได้
1. เอาสิ่งที่ลูกค้าอยากรู้เป็นตัวตั้ง แทนที่จะพูดแต่เรื่องที่เราอยากพูด
2. หาจุดเด่นที่คนอื่นไม่มี ผสมกับความชอบของผู้ชม ไม่ต้องรวย หล่อ สวย เช่น ชอบแมว ไม่ชอบกินข้าวแกง พูดไทยปนอังกฤษ
3. ใส่ตัวตนเข้าไปในคอนเทนต์ ทำซ้ำเรื่อยๆ ให้คนจำตัวเราได้

ของผมคือ "ปัน นักปรึกษาการขายที่ชอบส้ม" คือ expertise + quirk ใช้ด้วยกันทุกคลิป จน `audience (กลุ่มผู้ชม)` จำได้

### Part 3: แต่งตัวก่อนลงสนาม

ผู้ใช้ TikTok ใช้เวลาแค่ 0.8-1.5 วินาทีตัดสินใจว่าจะดูคลิปต่อหรือปัดทิ้ง ทุกอย่างในโปรไฟล์ของคุณเลยต้องช่วยกันดึงคนไว้

รูปโปรไฟล์ ต้องชัดเจน เห็นหน้าแล้วรู้เลยว่าคุณเป็นใคร ทำอะไร หลีกเลี่ยงรูปส่วนตัวที่ไม่เกี่ยวข้องกับงาน เช่น รูปเที่ยวทะเลหรือรูปสัตว์เลี้ยง

Bio ต้องอ่านแล้วรู้ทันทีว่าคุณช่วยอะไรเขาได้ ตัวอย่าง
- ✅ "เซลล์ขายรถที่ดูแลลูกค้าจบสุด ส่งมอบแล้วยังติดต่อได้ 24 ชม."
- ✅ "ผมช่วยคนแก้ปัญหานอนไม่หลับ โดยไม่ใช้ยา"
- ❌ "เลี้ยงหมา รักการเที่ยว ชอบฟังเพลง" ไม่มี `value proposition (คุณค่าที่นำเสนอ)`

คลิปปักหมุด 3 คลิปแรก ให้เลือกคลิปที่
- เล่าเรื่องตัวคุณ (who you are)
- สื่อตัวตนที่ `unique (มีเอกลักษณ์)` (what makes you different)
- แสดงให้เห็นว่าคุณแก้ปัญหาอะไรให้ใคร (what you do)

### Part 4: เขียนบทคอนเทนต์ให้ตรงกับ keyword ของช่อง

ก่อนหยิบมือถือขึ้นมาถ่าย ต้องออกแบบคอนเทนต์ก่อนทุกครั้ง

5 คำถามที่ต้องตอบก่อนกด Record
1. Content นี้เป้าหมายคืออะไร (growth / nurture / convert)
2. `keyword (คำค้นหา)` หลักของคลิปคือคำไหน ที่ audience พิมพ์ search จริง
3. เปิดหัวยังไงให้หยุดคนดูใน 3 วิแรก (specific claim / counter-intuitive / direct question)
4. ทำยังไงให้คนดูมีเป้าหมายในการดูจนจบ (curiosity loop / promise / cliffhanger)
5. จบคลิปแล้ว audience จะรู้สึก/ทำอะไรต่อ (comment / save / follow / DM)

ตอบ 5 ข้อนี้ให้ครบก่อน แล้วค่อยเริ่มถ่าย

เทคนิคเปิด 3 วิที่ใช้บ่อยในช่องผม
- Specific claim: "ยอดขายคงที่ทุกปีนี่สัญญาณอันตราย"
- Counter-intuitive: "จ้างเซลล์ก่อนมีระบบ = `burn out (หมดไฟ)` ใน 3 เดือน"
- Direct question: "ร้านคุณตอบลูกค้าภายในกี่นาที"

### Part 5: วัด funnel ไม่ใช่แค่ยอดฟอล

Followers เป็น `vanity metric (ตัวเลขสวยๆ ที่ไม่ได้สร้างรายได้)` Metric ที่ปิดดีลได้จริงอยู่ใน funnel analytics

Target และวิธีแก้ per layer (จากช่องผมเอง)

| Metric | วัดอะไร | Target | ของช่องผม | ถ้าต่ำให้ปรับอะไร |
|---|---|---|---|---|
| Profile visit / view | คลิปดึงคนมา profile ไหม | ≥ 1% | 0.99% ✓ | CTA + hook profile |
| Share / view | คลิปน่า share ไหม | ≥ 1% | 1.55% ✓ | angle + relatable insight |
| Follow / profile visit | Profile convince ไหม | ≥ 10% | ~83% ✓ | bio + pinned clips |
| Engagement rate | คลิปทำงานไหม | ≥ 5% | 7.9% ✓ | hook + middle tension |
| DM / follower | Follower กลายเป็น lead ไหม | ≥ 0.3% | track ใน CRM | CTA ชวน DM |

ถ้า profile visit rate ต่ำกว่า 0.5% hook ดี แต่คลิปไม่ดึงคนสู่ profile ต้องเพิ่ม CTA ชัดเจน เช่น "save ไว้ก่อน"
ถ้า follow rate จาก profile ต่ำกว่า 50% hook ดึงคนเข้ามาแต่ profile ไม่ convince ให้ follow ต้องแก้ bio + pinned clips
ถ้า DM rate ต่ำกว่า 0.1% ของ followers content ไม่ชวนคุยต่อ ต้องเพิ่ม open-ended question ท้ายคลิป

### Part 6: เปลี่ยนคนดูเป็นลูกค้า (ลูก funnel ที่ปิดดีลได้)

Conversion funnel ที่ผมใช้กับช่องตัวเอง ตัวเลขปฏิบัติที่วัดได้ทุก layer
1. Comment ที่เกี่ยวกับปัญหา → reply เชิญมา DM (measure: DM/comment rate)
2. DM → template 3 คำถามเพื่อ `qualify (คัดกรองเบื้องต้น)` ใน 15 นาที (`budget (งบประมาณ)`, `timeline (กรอบเวลา)`, decision process)
3. Qualified → นัด `discovery call (นัดคุยเบื้องต้น)` 30 นาที ฟรี (measure: call/qualified rate)
4. Call → ส่ง `proposal (ใบเสนอราคา)` ใน 24 ชม. แบบ `fixed-price (ราคาตายตัว)` (measure: proposal/call rate)
5. Close หรือ `disqualify (ตัดออก)` ภายใน 7 วัน ไม่ไล่ยืด (measure: close/proposal rate)

ผมติดตาม 5 ตัวเลขนี้ใน Google Sheets ทุกสัปดาห์ ถ้า bottleneck อยู่ที่ layer ไหน ผมจะโฟกัสแก้ layer นั้น ไม่ใช่พยายามโพสต์คลิปเพิ่ม

## AI ใน production: ใช้ตรงไหนได้ ตรงไหนห้าม (B2B trust boundary)

B2B selling คือ trust economy. ลูกค้าเลือกคนที่รู้สึกว่า "ของแท้" ไม่ใช่ generic content ที่ AI generate มา

สำหรับเจ้าของ B2B: AI ช่วยได้เฉพาะ **pre-production + post-production** (งานหลังบ้าน) แต่ **production (งานหน้ากล้อง) ต้องเป็นตัวจริง**

| Task | ใช้ AI ได้ไหม | เครื่องมือ | เหตุผล |
|---|---|---|---|
| Keyword research (หา topic ที่ audience search) | ✅ ใช้ได้ | Claude, ChatGPT | งาน research ไม่เกี่ยว trust |
| Idea generation (brainstorm angle ใหม่) | ✅ ใช้ได้ | Claude (good for brainstorm) | Input ยังมาจากเรา |
| Script drafting (ร่างบทจาก bullet note) | ✅ ใช้ได้ (ต้อง edit หนัก) | Claude API | AI ร่าง แล้วเรา rewrite ให้เป็นเสียงเรา |
| Caption / hashtag (เขียน caption สั้นๆ) | ✅ ใช้ได้ | Claude | งาน template ซ้ำๆ |
| Hook variation (ทดลอง hook 5 แบบก่อนเลือก) | ✅ ใช้ได้ | Claude | ช่วย iterate |
| Subtitle / transcribe (ทำ subtitle ไทย) | ✅ ใช้ได้ | TikTok auto / Whisper | งาน mechanical |
| **Video generation** (AI VDO) | ❌ **ห้าม** | — | B2B ต้องเห็นหน้าจริง เสียงจริง ไม่งั้น trust พัง |
| **Image generation** (AI avatar/thumbnail) | ❌ **ห้าม** | — | Generic AI image = signal ว่าไม่ authentic |
| **Voice cloning** | ❌ **ห้าม** | — | ลูกค้าดูออกทันที deal พัง |
| **AI talking head** (เช่น HeyGen) | ❌ **ห้าม** | — | B2B buyer รู้ทันที = lose trust forever |

กฎง่ายๆ: AI ช่วย **คิดกับเขียน** (research, draft, ideate) ได้ แต่ **หน้ากับเสียง** ต้องเป็นเจ้าของจริงทุกคลิป

ช่องผมเองใช้ Claude API draft script จาก bullet note + keyword research ก่อนถ่าย 80% ของ workflow แต่ทุกคลิปถ่ายด้วย iPhone + ring light ฿800 เสียงจริง หน้าจริง

## 5 ข้อผิดพลาดที่เจ้าของ SME ทำตอนเปิดช่องใหม่

1. เปิดช่องแล้ว `copy (ลอกเลียน)` คนดังโดยตรง ไม่มี `identity (ตัวตน)` ของตัวเอง
2. โพสต์ขายของตั้งแต่คลิปแรก algorithm ไม่ push คนไม่ follow
3. ไม่มี `content calendar (ปฏิทินคอนเทนต์)` ไม่มี `pillar (แกนหลักของคอนเทนต์)` ชัด โพสต์ตามอารมณ์ล้วนๆ
4. ด่วนตัดสินว่าช่องไม่เวิร์คใน 2 อาทิตย์แรก (เร็วเกินไป เพราะ algorithm ใช้เวลาทดสอบ 3-6 สัปดาห์)
5. ใช้ AI generate video/thumbnail เพื่อ scale (B2B buyer ดูออก trust พังถาวร)
