ที่ปรึกษาด้านการขายเก่งๆ มีเยอะ แล้วจะเลือกคนที่ใช่กับธุรกิจเรายังไง? ดู 3 อย่างนี้ครับ

## 1. Background และลักษณะลูกค้าของที่ปรึกษา

เช็คก่อนเลยว่าเขาเคยทำงานกับธุรกิจใน industry ของเราไหม

บางคนอาจจะเก่งจริง แต่ถ้าไม่เข้าใจธรรมชาติของธุรกิจเรา การใช้วิธีที่เขาถนัดก็อาจไม่เห็นผล

ตัวอย่างที่ผมเจอบ่อย: ที่ปรึกษาที่โตมาจาก B2C retail แล้วเอาวิธีคิด "เซลล์ Skill ดี ปิดด้วย urgency + ลดราคา" มาใช้กับ B2B วัสดุก่อสร้างที่ deal cycle 3-6 เดือน ลูกค้าตัดสินใจเป็นทีม และส่วนลดไม่ใช่ปัจจัยหลัก ผลคือทีมขายเหนื่อยขึ้น กำไรก็น้อยลง เพราะใช้วิธีผิด context

**วิธีถามคัดออก**
- "Client ล่าสุด 3 รายของคุณอยู่ใน industry อะไร ขนาดธุรกิจเท่าไหร่"
- "วิธีที่ใช้กับแต่ละเจ้าเหมือนกันหมดไหม หรือปรับตาม industry"
- ถ้าเขาตอบว่าใช้วิธีเดียวกันหมด หรือยกเคสที่ไม่ใกล้เคียงเราเลย แสดงว่าอาจจะยังไม่ใช่

## 2. Style การทำงานต้องเข้ากับ Culture องค์กร

จะรู้ได้ยังไง? ผมแนะนำให้ลองใช้บริการแบบ 'รายครั้ง' ก่อน เพื่อให้รู้ว่า style หรือ 'จริต' ในการทำงานเข้ากันได้ไหม

บางคนอาจจะส่งทีมงานมาแทน บางคนชอบสอน บางคนเน้นลงมือทำ workshop ต้องหาคนที่เหมาะกับเราให้เจอก่อน

ตัวอย่างจริง: ลูกค้าผมรายหนึ่งทำธุรกิจครอบครัว ทีมส่วนใหญ่อยู่กันมา 10+ ปี culture คือค่อยๆ คุย ค่อยๆ เปลี่ยน ถ้าไปจ้างที่ปรึกษา style "ฉีกระบบใหม่ทันที" ทีมจะต่อต้านทันที แต่ถ้าเป็น startup ที่อยากเปลี่ยนเร็วๆ ที่ปรึกษาแบบ "ค่อยเป็นค่อยไป" ก็จะถูกมองว่าช้าเกินไป

เจอคนที่ไม่ใช่ในครั้งแรกก็ไม่เป็นไร แค่หาคนใหม่ ไม่ได้แปลว่าที่ปรึกษาทุกคนจะแย่เหมือนกัน

**วิธีทดสอบ Style ก่อนจ่ายก้อนใหญ่**

จ้าง 1 session ราคา ฿15,000-30,000 (เช่น workshop 1 วัน, audit, หรือ discovery call) แล้ววัด 3 อย่างนี้
- เขาฟังก่อนพูด หรือมาถึงก็บอกคำตอบทันที
- งานหลัง session นำไปใช้ได้จริงไหม หรือเป็นแค่ roadmap กว้างๆ
- ทีมของคุณรู้สึกยังไงหลังคุยกับเขา

## 3. ที่ปรึกษาที่ดีต้องให้ Next-Step ชัดเจน

เขาต้องบอกได้ว่าคุณต้องทำอะไรต่อ ทำแบบไหน และวัดผลอย่างไร ไม่ใช่แค่มาพูดให้กำลังใจหล่อๆ แล้วกลับไป

มาตรฐานที่ผมใช้กับ client คือ ทุก session 1 ชั่วโมง ต้องมี action item 5-7 ข้อส่งให้ภายใน 24 ชั่วโมง พร้อม success metric ของแต่ละข้อ ถ้าที่ปรึกษาคนไหนทำไม่ได้ แปลว่าเขายังไม่มี framework ในหัวที่ชัดเจนพอ

## 4. 5 Red Flag ที่ควรเดินหนีตั้งแต่รอบแรก

1.  การันตียอดขายเพิ่มเป็น % แน่นอน ไม่มีใครการันตีได้จริง คนที่กล้าพูดแบบนี้คือคนขายฝัน
2.  คิดเงินเป็น % of revenue โดยไม่มีเพดาน แบบนี้เปิดช่องให้เก็บเงินไม่จบไม่สิ้น
3.  Case study มีแต่ brand ใหญ่ระดับชาติ ซึ่งไม่เกี่ยวอะไรกับ SME ที่มีงบ 30-200 ล้านบาทเลย
4.  ตอบคำถามด้วย buzzword เช่น "ขับเคลื่อน", "ครอบคลุมทุกด้าน", "อย่างครบวงจร" คนมีของจริงเขาไม่พูดกันแบบนี้
5.  เสนอ solution โดยไม่ขอดูข้อมูล sales pipeline หรือทีมงานของคุณก่อน แปลว่าเขาเตรียมสูตรสำเร็จมาขาย ไม่ได้สนใจแก้ปัญหาให้คุณจริงๆ

## ลองเล็กก่อน: วิธีทดสอบก่อน Commit Project ใหญ่

ก่อนจ่ายค่า project ฿100,000+ ให้ทำแบบนี้ก่อน:

1.  ขอ workshop 1 วัน ราคา ฿20,000-30,000 เพื่อดูวิธีทำงานจริง
2.  ระหว่าง workshop ให้วัดว่าเขาฟังก่อนพูดไหม และ deliverable ที่ได้วันนั้นใช้ได้จริงหรือเป็นแค่ roadmap เปล่าๆ
3.  ถ้าผ่าน 2 ข้อนี้ ค่อยคุยเรื่อง scope project ใหญ่
4.  ถ้าทำ workshop แล้วยังรู้สึกว่าไม่ใช่ ก็แค่หาคนใหม่ ค่า workshop ฿20,000 คุ้มกว่าจ่าย ฿200,000 แล้วต้องมาล้มโปรเจกต์กลางทาง
